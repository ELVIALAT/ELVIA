## 2. Análisis del Embudo (Funnel)

### A. Landing Page: La Fachada Premium
*   **Visuales**: Uso acertado de gradientes oscuros y Bento Grids (estética Apple/SaaS Moderno). Genera confianza inmediata.
*   **Curiosity Gap (Estrategia Maestra)**: El simulador interactivo permite a los usuarios "probar" la IA sin compromiso. El **Auth Wall** difuminado al final es una táctica de *Lead Gen* muy potente ("Tienes 4 errores críticos... regístrate para verlos").
*   **Prueba Social**: Se utilizan "Trust Badges" (100% ATS-Perfect), lo cual es vital para un producto de carrera.

### B. Registro y Onboarding
*   **Barrera de Entrada**: Baja. "2 análisis gratis" y "Sin tarjeta de crédito" son ganchos efectivos.
*   **Onboarding Detallado**: El formulario de 5 minutos es exhaustivo. 
    *   *Positivo*: Captura datos valiosos para personalizar la experiencia (salario, aspiraciones).
    *   *Riesgo*: Puede ser un punto de fricción (churn) si no se comunica bien que es "solo una vez". El autocompletado desde el CV mitiga esto brillantemente.

### C. Conversión a Pago (Pricing)
*   **Anclaje de Precios (Price Anchoring)**: El plan Pro Trimestral (MXN 699) hace que el mensual (MXN 299) parezca la opción lógica para una búsqueda rápida, pero el ahorro del 22% empuja al compromiso a largo plazo.
*   **Mecánica de Pago (Beta)**: Actualmente manual vía WhatsApp/Email.
    *   *Fortaleza*: Permite un cierre de ventas asistido y personal.
    *   *Debilidad*: No es escalable y puede enfriar el "impulso de compra" nocturno.
