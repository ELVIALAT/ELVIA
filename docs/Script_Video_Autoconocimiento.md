# Guion de Video: Proyecto Laboral y Autoconocimiento — ELVIA®

Este documento contiene el script estructurado para la generación automática de video en **Google Vids, Google Studio** u otra herramienta de edición audiovisual. 

---

## ℹ️ Instrucciones de Copiado a Google Docs
> [!TIP]
> **Google Docs cuenta con soporte nativo para Markdown.** 
> Para llevar este guion a Google Docs con formato perfecto (tablas, títulos, viñetas y negritas ya convertidas):
> 1. Selecciona todo el contenido de este archivo y cópialo (`Ctrl + C`).
> 2. Abre un documento en blanco en [Google Docs](https://docs.google.com).
> 3. Asegúrate de tener activa la opción en: **Herramientas > Preferencias > Detectar Markdown automáticamente** (suele estar activa por defecto).
> 4. Pega el contenido (`Ctrl + V`). El formato se autogenerará de inmediato con diseño premium.

## 🎬 Configuración General del Video
* **Público Objetivo**: Usuarios / Colaboradores B2B y B2C que ingresan por primera vez.
* **Estilo Visual**: Estética limpia tipo Apple (fondos color hueso `#F5F1E6`, acentos azafrán `#D97706` y azul profundo, animaciones fluidas, transiciones de cursor con zoom digital).
* **Tono de Voz**: Profesional, motivador, inspirador y estratégico.
* **Duración Estimada**: 90 segundos.
* **Configuración del Locutor IA**: Voz de perfil **"Professional / Executive"** en español neutro, con velocidad de reproducción a **1.0x** para transmitir calma y enfoque.

## 🎨 Recomendaciones de Producción para Google Studio:
* **Indicación para stock de video (AI prompts)**: *"Modern, bright minimalist office spaces, high-end Apple-style product shots, software interfaces with clean typography, pastel gradients, and abstract geometric diagrams."*
* **Paleta de Colores**: Utiliza el tono Hueso (`#F5F1E6`) para fondos, Negro Carbón (`#0E0D0A`) para los textos y Azafrán (`#D97706`) o Emerald (`#10B981`) para destacar elementos interactivos y botones principales.
